# Space Explorer

## Project Description
This is a Flask web application that displays space related information using APIs.

## Technologies Used
- Python
- Flask
- HTML
- CSS

## How to Run the Project

1. Open Command Prompt (CMD).
2. Navigate to the project folder.

Example:
cd space_explorer

3. Run the following command:

python app.py

4. Open the browser and go to:

http://127.0.0.1:5000/

The application will start running locally.
